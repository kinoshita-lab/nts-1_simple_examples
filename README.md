使い方:

ここのソースコードをダウンロードします。
[ダウンロードリンクはこちらです。](https://github.com/kinoshita-lab/nts-1_simple_examples/archive/master.zip)

できたフォルダを　logue-sdk/platform/nutekt-digital に丸ごと置きます。

それぞれのフォルダにあるソースコードをコンパイルして、NTS-1に書き込んで使います。
